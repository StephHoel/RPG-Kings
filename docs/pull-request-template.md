# Pull Request Template

Use este template para documentar e realizar um novo Pull Request.

Neste bloco, faça um breve resumo (1-2 linhas) do que este PR traz.

Copie este conteúdo ao criar um novo PR no GitHub.

---

## Versão

- Versão (SemVer): X.Y.Z
- Data: YYYY-MM-DD

---

## Destaques

- Nova feature 1
- Nova feature 2
- Item de bugfix
- Pequena melhoria 1
- Pequena melhoria 2

---

## Breaking Changes

- Se houver, documente claramente quais são as mudanças incompatíveis e o que os usuários devem fazer para migrar. Se não houve, apague este bloco.

---

## Checklist

- [ ] `docs/CHANGELOG.md` atualizado com a seção desta versão.
- [ ] `package.json` atualizado (se aplicável).
- [ ] Política de versionamento seguido.

---

Assinatura: @seu-usuario (adicione quem publicou)
